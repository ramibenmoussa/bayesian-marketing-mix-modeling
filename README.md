# Bayesian Marketing Mix Modeling (MMM)

This repository contains a complete Bayesian Marketing Mix Model (MMM) built
using PyMC. It includes a Jupyter notebook, test dataset, channel-level ROI
outputs, and a written report.

## Contents
- `MMM_Bayesian_MMM_Template.ipynb` — main notebook
- `MMM_test_data.csv` — input data (synthetic)
- `roi_by_channel.csv` — computed ROI by marketing channel
- `Report.pdf` — final summary report
